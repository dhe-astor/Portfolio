# 🌐 Personal Portfolio

Welcome to my personal portfolio website project! This project showcases my skills, projects, and experience as a Computer Science student.

---

## 📌 About the Project

This is a simple and responsive portfolio website built using:

* HTML5
* CSS3
* JavaScript (if used)

The goal of this project is to present my work in a clean and professional way and to practice frontend development skills.

---

## 🚀 Features

* Clean and modern UI design
* Responsive layout (works on mobile & desktop)
* About Me section
* Projects showcase
* Contact section

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
